天色渐渐暗淡，窗户被夜色染上墨，白色桌面在灯光下显得昏黄。大部分人早已下班回家，环顾四周，大多是空荡无人的椅子，只有我偶尔敲击键盘的清脆声响，窗外的漆黑环绕着整栋大楼，我喜欢这种感觉——寂静之中，我感受到自己存在的真实，我处理完工作，舒展了一下四肢，看了看右下角的时间，十一点多，这是一天临近终点的时刻。下班骑车回家，登上自行车，链条和齿轮的运动在无声的夜中十分清晰，还有车轮驶过车道的声音，记忆中有许多我在凌晨骑车的印象，这说明我总是在夜中骑车，那些夜晚总是很凉爽，或者说每个夜晚都很凉爽，我穿过黑色凉意的洗刷，在床上躺下，在彻底的黑暗中入睡。死亡是凉爽的夜晚，我想到了这句话，它带着结束一切的安宁和平静，恰如睡觉的状态，这句话我从余华的书中得知，好像是海涅写的。

最近在创作一部小说，我如何绞尽脑汁，也无法让笔下的文字追赶上想象中的画面，笔下人物那木偶一般的感觉让我感到沮丧。关于夜的描述，记忆中有一篇绝妙的文章，那就是《鲁迅选集》中的那篇《夜》。尽管如此，我仍然渴望自己创造感受中的漆黑，渴望能将黑夜描绘得恰到好处，因为我常独行在夜中，感到自己独一无二的感受逐渐消失在夜中。

梦中，我在店铺前，看着老师傅流利地制作一份巨大的肉烧饼，充满期待，外面下起了雨，我可能会迟到，尽管我不知道自己要去哪里，但眼前这是一份比头还大的饼，或者说已经不能称为烧饼了，而更像是个巨型面皮包着的一整块牛肉，我并不知道它是牛肉还是猪肉，渴望告诉我它是牛肉，我觉得它值得我等待，下一刻，我又在楼房的三楼往下望去，或者说是四楼，天气晴朗，楼下有许多人，学生打扮，好像正是我的目的地学校，原来我是一名学生，然后我的手机收到了同学发来的消息，是一张聊天记录的截图，数学老师提到我的缺课，除此之外什么都没有，看来老师也清楚，旷课是在算不上什么大事。

我的床没有枕头，我一般平躺着，或者侧着身，睁开眼，梦就这样中止了。
