自由幻想

    • 笔记
    • 关于
    • 回声

2020-03-26

写在人生中情绪起伏的时刻，某一天，你会想起她，而这里，拥有这短暂的纪念。

她是你的大学同学，昨天和她聊天，你意识到这样的交流对于此刻的你是多么珍贵，眼前的的现状很糟糕，让你精神恍惚，怀疑现实，你不打算再想这些破事情，于是倾注了所有精神和她交流。

你对此产生了微弱的期盼，像是在茫茫黑夜中发现了火光，生活此刻燃起希望，你想对她倾诉，不过好像没有对她说出口的理由，这些事情发生在谁身上说出口都是糟糕的精神负担。那一刻心跳地很快，你开始想幻想未来。下意识你就听到了内心中的声音，安静地待在房间里，等待着糟糕的一切过去。过了这段时间，所有事情成为过去，但你对此印象深刻，可能记上六个月，可能记上永远。

你是如此的片面和幼稚。时间流逝，记忆有一天会消失殆尽，那一刻悸动的感觉，难以被解释，可以称为缘分，她说了鼓励的话，至少在你看来算得上，让你难忘。想着自己拿着手机看着聊天记录傻笑，有点可笑，于是它仅存你的心中。

你不再去想，只是向前走，你知道它会结束，只是早一点晚一点的问题。生活时刻在变化，包括你。

某天，你又想到了她，她在干什么，她的一天是否也像你这般虚度。你想她理应有许多朋友，那生活自然不会像你这样简单。对未知的好奇让你想问一问这个问题，但终还是没有说出口，这不符合你的性格。她真有趣，有趣是对人的一个多么高的评价，有人生来拥有，然后失去。或许这只是你的一厢情愿。只是因为她长的好看。也有可能因为好奇当下的交流为何发生，让你有种因缘际会的感觉，她笑起来真可爱。

此刻，这段时间，当下，夜晚六点，你频繁的想起她，你不知道为何。这种状态，你不知道怎么形容，你很想摆脱这种状态，越是挣扎越是陷入其中，你思前想后，突然有点不舍，我看这是你转移注意力的一种手段，目的是为了暂时忘记不愉快的事情。在这里，你想要呼喊，却又只能沉默着。你记录下这些想法，渴望在未来感受到一些真实，你为什么要想起她，你同很多人这样聊过，然后又没有然后，你确信自己以后会一个人生活，带着一副热烈又寻常的模样，你却又做不到行动与想法一样实际，这个世界习惯彰显生活的美好一面太久，你却如此消极。

你总是沉默着，好像若有所思，没人知道，大半的时间里你脑子里空荡荡，什么也没有想。你找不到，这个问题能有什么好的解决方法，关于这个问题的描述你也给不出，你甚至在怀疑这到底算不算一个问题。你不知道这样想有什么好处，也可以说有什么用，但是你避免不了去想，去意识到它的存在。

你想不这么矛盾，却又无法控制自己的想法，唯有当几件事情，在你的脑子里碰撞在一起然后炸裂，什么都回想不起来的时候，你才能享受片刻的宁静，你不想患得患失，认为解决它的办法就是减少发生的可能，你开始走向不同。你认为人生中应该减少一些仪式感和新鲜感，这两样东西被使用地太频繁，对你来说不再珍贵，有时候又不得不通过它们来给自己的内心一些安宁和精神动力。你想到了你自己，你对自己以后要成为一个什么样的人自信满满，对一些人一些事充满怀疑，现在你却发现自己在改变，问自己，回应只有沉默，你只能相信有些东西无论如何也不会变化。

你在路上，背影潇洒或者有点落寞，夜晚安静无声，街道上少有行人，四周唯有寂静和黑暗。路灯很亮，你的影子被拉得很长，在这个脱离生活的凌晨几分，感受时间很慢，感受到自己的虚弱，你感受到所有的悲伤愤怒都会趋于平静，难逃过忽略再被遗忘。

人要为自己的生活构建几个不同的世界，一个世界的信仰破碎了，还要有其他世界的希望支撑，不至于被痛苦淹没太久。
