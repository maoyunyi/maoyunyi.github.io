2020/02/02日 Github 为所有的公共库生成了快照，以QR码的方式存在186卷胶卷里，每卷长达1066米，一共写入了21TB的仓库数据于07/08日成功地储存在北冰洋挪威群岛的斯瓦尔巴群岛地下数百米深的永冻土内，预计可以存储一千年以上。

好艺术。
